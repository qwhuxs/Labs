# Privacy Policy

## 1. General Information

This project (Anagram Game) does not collect or process personal user data.

## 2. Cookies

The application uses cookies and/or localStorage only to store user preferences and consent decisions.

No tracking systems, analytics services, or third-party advertising tools are used in this project.

## 3. Data Storage

All data is stored locally in the user's browser and is not transmitted to any external servers.

## 4. User Rights (GDPR)

According to GDPR principles, users have the right to:

- Accept or decline cookies
- Clear stored data manually in their browser
- Stop using the application at any time

Since no personal data is collected, no additional data processing is performed.
